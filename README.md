# Daily Expense Tracker

## Overview

Daily Expense Tracker is a web application developed using Python and Django to help users manage and monitor their daily expenses efficiently. The application allows users to add, update, and delete expense records while tracking spending categories and summaries.

## Features

* Add daily expense records
* Update and delete expenses
* Category-wise expense tracking
* Expense summary and calculation
* User-friendly interface
* Secure database management using Django

## Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* SQLite

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/daily-expense-tracker.git
```

2. Navigate to the project folder:

```bash
cd daily-expense-tracker
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the server:

```bash
python manage.py runserver
```

5. Open in browser:

```text
http://127.0.0.1:8000/
```

## Project Structure

* `tracker/` – Main application files
* `templates/` – HTML templates
* `static/` – CSS and JavaScript files
* `db.sqlite3` – Database file
